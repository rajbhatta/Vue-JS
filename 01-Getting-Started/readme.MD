## Getting Started with VueJs ##

### 1. Installing VueJs ###
```js
npm install -g @vue/cli
```

### 2. Creating VueJs Project using command ###
```js
vue create my-project
# OR
vue ui
```

### 3. Creating VueJs Project using UI ###
```js
vue ui
```

### 4. Running Vue project using CLI ###
```js
npm run serve
```
